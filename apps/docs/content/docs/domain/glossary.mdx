<Glossary>
  <Term term="units" ko="수량" code="posting.units_minor">
    계정 상품(commodity)으로 기록된 사실량.
  </Term>
  <Term term="weight" ko="장부가액" code="posting.weight_minor" notTo="units를 잔액으로 쓰지 말 것">
    같은 사건을 기능통화로 환산한 값. 잔액 집계는 Σ weight만 사용.
  </Term>
  <Term term="functional currency" ko="기능통화" code="book.functional_currency">
    `init --currency`로 정한다. booking commodity.
  </Term>
  <Term term="weight_source" code="identity|actual|rate|plug">
    기능통화 다리는 identity. 관측 총액은 actual, 환율 유도는 rate, 잔차 다리는 plug.
  </Term>
  <Term term="draft" ko="초안" code="txn.status=draft" notTo="posted">
    검토 전 거래. 잔액·리포트·마감 게이트에서 제외.
  </Term>
  <Term term="posted" ko="전기됨" code="txn.status=posted">
    승인·직접 기표된 거래. 장부의 사실.
  </Term>
  <Term term="assertion" ko="잔액 대사" code="balance_assertion">
    외부 명세서와 잔액 대사. `assert`. 마감 게이트 입력.
  </Term>
  <Term term="period" ko="마감 기간" code="period.status">
    close_grain(기본 month). open / closed / locked.
  </Term>
  <Term term="cash" ko="현금성" code="account.cash" notTo="Assets:Bank:* prefix">
    현금흐름 opening 집합 플래그. 코드 prefix가 아님 (ADR-004).
  </Term>
  <Term term="billing cycle" ko="청구주기" code="card.cycle_close_day">
    카드 청구·결제 규칙. close-day, payment-day, offset.
  </Term>
  <Term term="schedule" ko="스케줄">
    카드/할부/정기/대출 예측 행. 저널 밖. cashflow가 투영.
  </Term>
  <Term term="checkpoint" ko="WAL checkpoint" code="holiday checkpoint" notTo="snapshot.kind=checkpoint">
    SQLite WAL을 ledger.db로 fold. 스냅샷 checkpoint와 무관.
  </Term>
  <Term term="audit chain" ko="감사 체인" code="audit_log">
    해시 체인. `verify`가 검사.
  </Term>
</Glossary>

## 계정 코드

Beancount 스타일 `Root:Segment:…`. Root ∈ Assets/Liabilities/Equity/Income/Expenses.  
예: `Assets:Bank:KB:Checking`, `Liabilities:Card:Shinhan:Installment`, `Income:FX:Unrealized`.
