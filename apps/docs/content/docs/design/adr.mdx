<Decision
  id="ADR-001"
  status="locked"
  title="잔액은 저장된 weight"
  date="2026-07-01"
  rejected={[
    { option: '조회 시 rate×units로 재균형', because: '과거 기표가 환율 변경에 흔들리고 허용오차가 생긴다' },
    { option: 'Beancount currency groups', because: '개인 가계 원장에 과도하고 손실성' },
  ]}
>
  Σ `weight_minor`가 균형이다. 조회 시 rate×units로 다시 맞추지 않는다. 허용오차 없음.
</Decision>

<Decision
  id="ADR-002"
  status="accepted"
  title="SoR = 로컬 SQLite + 해시 체인"
  date="2026-07-01"
  rejected={[
    { option: 'JSONL + git을 SoR로', because: '동시성·부분 쓰기·에이전트 idempotency에 약하다' },
    { option: 'CLI에서 Postgres 직접 오픈', because: '1인 로컬 제품 경로를 흐린다' },
  ]}
>
  `.holiday/ledger.db`가 진실. 감사는 audit_log 체인. CLI는 Postgres를 열지 않는다.
</Decision>

<Decision
