## 토폴로지

```
@holiday/core
  ↑
@holiday/store-sql          (단일 LedgerStore 구현)
  ↑
@holiday/store-sqlite | @holiday/store-postgres
  ↑
@holiday/cli                (composition root → sqlite only)
```

`assertEngineTier`는 atomicMultiRowWrite · uniqueConstraints · readAfterWriteConsistency를 요구한다.  
store-testkit 컨포먼스 **18** 케이스.

## 네 링

1. Domain (`Txn.create`)
2. Port (`appendTxn(ValidatedTxn)`)
3. Storage triggers (11)
4. Audit hash chain (`CHAIN_HASH_VERSION = 1`)

## 배포

- 플러그인 커밋 번들: `plugin/bin/holiday.mjs`
- 런타임: Node ≥ 24, `node:sqlite`
