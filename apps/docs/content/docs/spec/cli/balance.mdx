## JSON (`--json`)

```json
{
  "asOf": "2026-07-31",
  "balances": [
    {
      "accountId": "…",
      "accountCode": "Liabilities:Card:Shinhan",
      "commodity": "KRW",
      "unitsMinor": "-1240000",
      "weightMinor": "-1240000"
    }
  ],
  "liabilityMaturity": {
    "asOf": "2026-07-31",
    "currentMinor": "1240000",
    "nonCurrentMinor": "0",
    "totalMinor": "1240000",
    "byAccount": [
      {
        "accountId": "…",
        "accountCode": "Liabilities:Card:Shinhan",
        "currentMinor": "1240000",
        "nonCurrentMinor": "0",
        "totalMinor": "1240000"
      }
    ]
  }
}
```

- `balances[]` 행의 `unitsMinor` / `weightMinor`는 **문자열** bigint (기존과 동일 의미).
- `liabilityMaturity` 금액은 **빚의 크기(절댓값)** 문자열. 계약:
  [SPEC-liability-maturity](/docs/domain/liability-maturity).
- 조회 범위에 부채 잔액이 없으면 `liabilityMaturity`는 `null`이다.
- 0.4.2 이전은 잔액 배열만 stdout에 냈다. 파서는 `balances` 키를 본다.

## 사람 출력

계정 행 뒤에 부채가 있으면 한 줄:

`유동부채 … · 비유동부채 … · 합 … (기준일 YYYY-MM-DD)`
