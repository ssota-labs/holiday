## `--data-file` 스키마

금액은 **소수 문자열**. `number` 금지.

- `lines` 최소 1개, `kind` ∈ `national_pension` · `health_insurance` · `long_term_care`
- 같은 `kind` 중복 금지
- 직장 공제액이 아니라 **직접 납부** 달만
- CLI는 요율로 금액을 검증·수정하지 않는다
