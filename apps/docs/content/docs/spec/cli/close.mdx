## JSON — dry-run

기존 키에 `liabilityMaturity`만 추가한다. 기준일은 그달 말일.

```json
{
  "month": "2026-06",
  "wouldRevalue": 0,
  "assertions": 2,
  "dryRun": true,
  "liabilityMaturity": {
    "asOf": "2026-06-30",
    "currentMinor": "4200000",
    "nonCurrentMinor": "86000000",
    "totalMinor": "90200000",
    "byAccount": []
  }
}
```

부채 잔액이 없으면 `liabilityMaturity`는 `null`.
`holiday balance --as-of <말일>`의 합계와 같다
([SPEC-liability-maturity](/docs/domain/liability-maturity) 수락 4).

```json
}
```
