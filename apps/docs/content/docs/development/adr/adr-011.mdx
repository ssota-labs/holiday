    { option: '보험 비용 전표만으로 충분하다', because: '낸 돈은 전표로 남는다. 다만 “언제 직장/지역이었나”, “지역이던 달 고지액”은 전표 다리에 안 들어간다.' },

    { option: 'tax_return에 사회보험을 섞는다', because: '기간 단위·칸·개정 사유가 다르다.' },

  ]}
>
사회보험은 저널 밖 사실 두 층이다.

1. `insurance_enrollment` — 보험 종류·자격·시작일·종료일
2. `insurance_contribution` / `_line` — 직접 납부 달의 부과연월·항목 금액

직장 공제액은 `income_settlement`가 담당한다. 현금 납부는 보험 비용 전표다.
자격·고지 기록은 전표를 만들지 않고, 요율로 금액을 다시 계산하지 않는다.
</Decision>
