월세·넷플릭스처럼 **아직 안 온** 반복 지출만 현금흐름에 올린다. 오늘·과거분은
이미 장부에 있다고 보고 다시 세지 않는다.

## 정기지출 한 건에 들어가는 정보

| 항목 | 의미 |
|---|---|
| 금액·주기 | 매달 얼마, 언제 |
| 결제 수단 | 통장이면 발생일 현금 / 카드면 청구 결제일 현금 |
| 메모 | 월세, 구독명 등 |

## 수락 기준

1. 오늘 이하 발생분은 투영하지 않는다 ([POLICY-005](/docs/domain/policy#POLICY-005))
2. 카드로 내는 정기지출의 현금 유출일은 청구 결제일이다
   ([POLICY-010](/docs/domain/policy#POLICY-010))
3. 시작일이 오늘보다 늦어도, 시작일 이후 미도래 회차는 현금흐름에 잡힌다
   ([시작일 계약](/docs/domain/recurring-active-window))
4. 시작일 전 주기일은 현금흐름에 잡히지 않는다
5. 시작 예정 스케줄이 목록에 보인다
