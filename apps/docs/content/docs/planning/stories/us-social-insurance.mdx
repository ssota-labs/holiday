**As a** 1인 사용자  
**I want** 국민연금·건강보험 자격을 기간으로 남기고, 지역가입처럼 내가 직접
내는 달의 금액도 월별로 보고 싶다  
**So that** 직장↔지역이 바뀌어도 공단 PDF 없이 자격·납부 이력을 조회할 수 있다

### 무엇을 남기는가

자세한 칸은 [prd-social-insurance-sor](/docs/planning/prds/prd-social-insurance-sor)에 있다.

1. **자격 구간** — 보험 종류(건강보험·국민연금), 자격(직장·지역·임의가입),
   시작일·종료일, 메모  
   지금 직장가입자라 고지가 없어도 구간은 남긴다.
2. **직접 납부 월 금액** — 부과연월, 국민연금·건강보험·장기요양 등
   고지서·납부확인서에 **이미 적힌** 값  
   직접 내는 달에만 적는다. 직장 공제액은 여기 넣지 않는다.

금액은 다시 계산하지 않는다. 공단에 보이는 값을 그대로 옮긴다.

### 수락 기준

1. 자격 구간을 저장·조회한다. 같은 보험에서 열린 구간이 겹치면 **거절**한다
2. 자격이 바뀌면 이전 구간을 닫고 새 구간을 열 수 있다
3. 직접 납부 달의 표지+항목 금액을 **한 번에** 저장한다
4. 월 금액은 **고지서에 찍힌 값만**이다. 요율로 추정하지 않는다
5. 같은 부과연월·같은 차수를 다시 넣으면 **거절**한다. 정정은 **새 차수**다
6. 자격·월 금액 저장만으로 **전표가 생기지 않는다**
7. 직장가입자 구간만 있어도 된다 (금액 없이도 자격 이력만 관리)
